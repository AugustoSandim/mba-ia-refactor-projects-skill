# Project Analysis Heuristics

- Detectar stack via arquivos de dependência/imports.
- Detectar persistência por ORM/SQL raw.
- Inferir domínio por entidades/rotas.
- Classificar arquitetura atual (monolítica, pseudo-camadas, MVC parcial).

Resumo obrigatório:

- Language, Framework, Dependencies
- Domain
- Architecture
- Source files analyzed
- DB tables/collections
